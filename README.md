# Zero-Knowledge Encrypted Notes

## AI Usage

This project utilizes GenAI in a mostly read-only capacity for code reviews and planning. GenAI may also be used for some git commit messages, changelogs, and documentation. No generated code is used.

### IDE Extensions

Developpment is augmented using the [Continue.dev](https://www.continue.dev/) VSCode extension with local configurations and rules.

### Agents and Models

- **Chat/Planning:** [Anthropic Claude Sonnet 4.6](https://www.anthropic.com/claude/sonnet)
- **Embedding:** [Voyage Code 3](https://docs.voyageai.com/docs/embeddings)
- **Reranking:** [Voyage Rerank 2.5](https://docs.voyageai.com/docs/reranker)

### Rules and Prompts

See `.continue/` directory.
